# JWT Tools

`第一次发布程序，不知道要写啥`

`为了学习rust+tauri实现的一个图形化界面的JWT爆破工具，喜欢的话点一个Star吧~`

### 主要功能

* **JWT 令牌爆破**：可以自动解码输入的 JWT 令牌，并分别显示其头部（Header）和载荷（Payload）以及爆破密钥。

### 优势

* 不知道有什么优势
* Rayon 并行库使用工作窃取算法自动并行化，自动负载均衡。
